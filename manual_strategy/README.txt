indicators.py file has all the logic for generating graphs for 6 Technical Indicators.
TheoreticallyOptimalStrategy.py has all the logic for creating the optimal strategy.
maketsimcode.py has the exact same logic of the last project.

To run please copy the file into a folder which has the data and util functions, and then run:
1. PYTHONPATH=../. python indicators.py
This will generate plots for technical indicators.

2. PYTHONPATH=../. python TheoreticallyOptimalStrategy.py
This will generate plots for theoretically optimal strategy.
